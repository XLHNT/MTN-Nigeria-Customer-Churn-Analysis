## Data Directory

This folder contains the dataset and documentation for the MTN Nigeria Customer Churn Analysis.

Contents

- `MTN_Nigeria_Customer_Churn.xlsx` - Original dataset (50,000+ customer records)
- `data_dictionary.md` - Comprehensive column definitions and statistics
- `schema.sql` - Database structure and table definitions

## Dataset Overview

- **Total Records:** 50,000+ customers
- **Geographic Coverage:** All 36 Nigerian states + FCT
- **Time Period:** Historical snapshot with lifetime metrics
- **Target Variable:** Customer Churn Status (Yes/No)

## Data Privacy

This dataset has been anonymized for portfolio demonstration purposes. No personally identifiable information (PII) is included.

For detailed column specifications, see [data_dictionary.md](data_dictionary.md).
