# Query Results & Outputs

This folder contains exported results from SQL queries and aggregated metrics.

## Contents

### query_outputs/
CSV exports from each SQL query for:
- Reproducibility
- Further analysis
- Visualization input
- Documentation

### summary_metrics.xlsx
Excel workbook with:
- Key performance indicators
- Aggregated statistics
- Cross-tabulated results
- Executive summary tables

## Usage

These results can be used to:
- Verify analysis reproducibility
- Create additional visualizations
- Conduct further statistical testing
- Share findings with stakeholders

## Data Format

All CSV files include:
- Header row with column names
- Properly formatted numeric values
- Date fields in ISO format
- UTF-8 encoding
