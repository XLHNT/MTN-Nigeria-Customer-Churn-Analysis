# Visualizations

This folder contains charts, graphs, and dashboard screenshots from the analysis.

## Contents

- `dashboard_overview.png` - Main executive dashboard
- `churn_by_state.png` - Geographic heatmap of churn rates
- `churn_by_age_gender.png` - Demographic breakdown
- `tenure_lifecycle.png` - Churn by customer tenure stages
- `revenue_impact.png` - Financial analysis visualization
- `top_churn_reasons.png` - Attribution analysis chart

## Visualization Principles

All visualizations follow these principles:
- Clear, actionable insights
- Professional color schemes
- Labeled axes and data points
- Executive-ready quality
- Mobile-friendly formats

## Tools Used

- Microsoft Excel for chart creation
- Power BI for dashboard design

## How to View

All images are high-resolution PNG format and can be:
- Viewed directly on GitHub
- Downloaded for presentations
- Embedded in reports
- Shared with stakeholders
